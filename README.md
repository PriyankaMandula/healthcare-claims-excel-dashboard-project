# Healthcare Claims Data Analysis Dashboard (Excel Project)

## Project Overview

This project focuses on analyzing healthcare insurance claims data using Microsoft Excel. The objective is to clean raw data, perform exploratory data analysis, and build an interactive dashboard to derive business insights from healthcare claim records.

The project demonstrates core data analytics skills including data cleaning, data transformation, pivot table analysis, and dashboard creation.

## Tools & Technologies

* Microsoft Excel
* Data Cleaning Techniques
* Pivot Tables
* Data Visualization
* Dashboard Design

## Dataset Information

The dataset contains healthcare insurance claim records including claim details, approval status, processing time, insurance type, and city information.

Total Records: 1200+

## Data Cleaning Steps

* Removed duplicate records
* Trimmed extra spaces
* Standardized categorical values
* Converted timestamp fields to date format
* Validated numeric columns
* Created calculated columns for analysis

## Key Performance Indicators (KPIs)

* Total Claims
* Total Claim Amount
* Claim Approval Rate
* Average Processing Time

## Dashboard Components

The Excel dashboard includes the following visualizations:

* Claim Approval Distribution (Pie Chart)
* Claims by City (Bar Chart)
* Claims by Insurance Type (Column Chart)
* Monthly Claim Trend (Line Chart)
* Interactive Filters using Slicers

## Business Insights

* Majority of claims are approved compared to rejected claims.
* Claim volume varies across cities indicating regional healthcare utilization.
* Insurance type influences claim distribution.
* Monthly trends help identify claim processing patterns.
